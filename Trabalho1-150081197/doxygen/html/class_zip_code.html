Detailed Description

Classe ZipCode representa o CEP do usuario. O CEP pode ser: 1000000 a 5999999 e 8000000 a 8499999 (Sao Paulo), 20000000 a 26600999 (Rio de Janeiro), 70000000 a 70999999 (Brasilia), 40000000 a 41999999 (Salvador) e 60000000 a 60999999 (Fortaleza).
